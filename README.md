# Digital Forensics Examiner Roadmap
### UNC Charlotte — Computer Science + Criminal Justice Minor

> A structured, public learning roadmap built for the path to becoming a digital forensics examiner. Built on TryHackMe Premium. Documented openly so other students can follow the same path.

---

## Who This Is For

This repository serves two audiences:

**For students** — a complete, opinionated roadmap from zero to forensic examiner portfolio, with room-by-room guidance, project templates, and honest notes on what matters and what doesn't.

**For recruiters** — a transparent record of skills built, tools learned, and real investigative work completed. Every project in Phase 5 is a full forensic case with documented methodology, evidence handling, and professional reporting.

---

## Background

Rising sophomore, Computer Science major at UNC Charlotte pursuing a Criminal Justice minor. The CJ minor is not a checkbox — it directly shapes how I approach evidence handling, legal defensibility, and report writing. Most technically-trained forensics candidates don't have that foundation.

Target career path: digital forensics examiner in corporate litigation support, federal law enforcement, or a private forensics firm.

## Tools Covered

### Primary Examiner Tools
- **Autopsy** — Open-source disk forensics GUI
- **FTK Imager** — Forensically sound disk imaging (free, industry standard)
- **Volatility 3** — Memory forensics framework
- **KAPE** — Rapid artifact triage and collection (Kroll)
- **Magnet AXIOM** — Industry-leading commercial forensics platform (trial)
- **EnCase** — Commercial examination platform (trial)
- **Redline** — Endpoint memory and IOC analysis (Mandiant)

### Windows Artifact Tools (Eric Zimmerman Suite)
ShellBags Explorer · JLECmd · LECmd · MFTECmd · PECmd · RECmd · Registry Explorer · Timeline Explorer

### Network & Analysis
Wireshark · tshark · NetworkMiner · CyberChef · Zeek · Snort

### Malware Analysis
REMnux · FLARE-VM · Any.run · CAPA

### Case Management
TheHive · Obsidian (notes and documentation)
