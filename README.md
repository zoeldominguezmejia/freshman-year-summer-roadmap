## 📋 About This Roadmap

A structured, dual-track cybersecurity learning roadmap combining hands-on technical labs via **TryHackMe Premium** with professional certifications through the **UNCC Coursera Career Academy** (free).

By balancing technical proficiency — parsing memory dumps, disk forensics, log triage — with enterprise frameworks and recognized credentials, this roadmap prepares you for DFIR internship applications by the end of sophomore year.

## 🛠️ Tools You Will Learn

- **Autopsy** — Disk forensics GUI
- **Volatility 3** — Memory forensics framework
- **KAPE** — Rapid triage and artifact collection
- **Redline** — Endpoint memory analysis and IOC scanning
- **Velociraptor** — Enterprise endpoint monitoring and forensic response
- **TheHive** — Incident response case management
- **Wireshark / Tcpdump** — Network traffic and PCAP analysis
- **Splunk / Elastic (ELK)** — SIEM log analysis and alert triage
- **CyberChef** — Data decoding, encoding, and IOC extraction
- **Regripper / Eric Zimmerman Tools** — Windows registry and artifact forensics
- **Plaso / log2timeline** — Full system timeline creation
- **Nmap / Metasploit** — Scanning and exploitation basics
- **Burp Suite** — Web application testing
- **REMnux / FlareVM** — Malware analysis environments

## 💡 Sophomore Year Core Rules

1. **GPA first.** HR filters for GPA during sophomore internship screening. Technical skills matter, but a 2.5 GPA is an automatic disqualifier at many companies.
2. **5–7 hours per week, not more.** DSA and Linear Algebra are demanding. Protect your academic performance.
3. **One CyberDefenders challenge per week.** Keeps your DFIR skills active without heavy time investment. Write up every single one.
4. **Security+ by December.** Book the exam before you feel fully ready. A scheduled exam date creates accountability.
5. **Winter break is a sprint.** No classes, no lab. Use it to take Security+ if not done and start IBM cert courses 1–3.
6. **IBM cert by April.** Two hours per week on Saturdays gets you through 8 courses in one semester.
7. **Apply broadly and early.** Most summer 2027 internship applications open September–November 2026. Do not wait until spring.
8. **Join the 49er Cyber Club.** One warm intro from an upperclassman beats ten cold applications.
