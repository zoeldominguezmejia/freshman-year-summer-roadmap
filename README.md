## 📋 About This Roadmap

This is a structured, dual-track cybersecurity learning roadmap that seamlessly meshes hands-on laboratory technical skills via **TryHackMe (Premium)** with formal industry name-recognition through the **UNCC Coursera Career Academy**. 

By balancing technical proficiency (parsing logs, memory analysis) with high-level corporate security frameworks, this timeline prepares you for enterprise DFIR internships by the end of your sophomore year.

**Platforms:** [TryHackMe Premium](https://tryhackme.com) & [UNCC Coursera Career Academy](https://www.coursera.org/career-academy/programs/career-center-skill-pathways-program-dddnz?authProvider=charlotte)
**Primary Goal:** Digital Forensics & Incident Response (DFIR) Portfolio
**Credentials Targeted:** Google Cybersecurity Professional Certificate, IBM Cybersecurity Analyst Certificate, CompTIA Security+
