## ⭐ Priority Rooms to Slow Down On
 
**Active Directory Basics** — AD is the backbone of every enterprise Windows environment. You will encounter it in every internship. Take notes on users, groups, GPO, and domain controller concepts.
 
**Digital Forensics Fundamentals** — This is the first time DFIR appears in structured form. Go slow. Every artifact type introduced here gets a full phase later.
 
**Incident Response Fundamentals** — Learn the IR lifecycle cold: Preparation → Identification → Containment → Eradication → Recovery → Lessons Learned.
 
**CyberChef: The Basics** — You will use CyberChef constantly in DFIR for decoding strings, extracting IOCs, and analyzing encoded payloads. Master the basics now.

## 🔗 Supplemental Resources
 
- **Web Exploitation Sandbox:** [PortSwigger Web Security Academy](https://portswigger.net/web-security) — Free, made by the Burp Suite team. The ultimate companion for Section 8.
- **Exploit Database:** [GTFOBins](https://gtfobins.github.io/) — Unix binaries that can be abused for privilege escalation. Bookmark this.
- **Nmap Reference:** [Nmap Scripting Engine (NSE)](https://nmap.org/book/nse.html) — Pre-written scripts for automated vulnerability checks during scans.
- **Windows PrivEsc Cheat Sheet:** [PayloadsAllTheThings](https://github.com/swisskyrepo/PayloadsAllTheThings/blob/master/Methodology%20and%20Resources/Windows%20-%20Privilege%20Escalation.md) — Commands to hunt for weak file permissions and unquoted service paths.
- **AD Visual Reference:** [Active Directory Explained](https://www.varonis.com/blog/active-directory) — Varonis's AD overview is one of the clearest free references available.
