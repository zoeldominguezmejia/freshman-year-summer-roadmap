# Phase 3 — SOC Level 1 Learning Path
**Weeks 5–7 | June 23 – July 13 | ~50 hours | ~17 hrs/week**

> Note: Overlaps with end of Phase 2 — start this as you finish Phase 2's final weeks.

---

## 🎯 Goal
Learn to operate as a Security Operations Center (SOC) Level 1 analyst. Understand how defenders monitor, triage, and respond to real incidents. This phase is critical for DFIR because it teaches you how forensic evidence plugs into real incident workflows.

---

## 📚 Learning Path: SOC Level 1
**Link:** https://tryhackme.com/path/outline/soclevel1

### Module Breakdown

| Module | Topics | Est. Time |
|--------|--------|-----------|
| **Blue Team Introduction** | SOC structure, analyst roles, how blue teams operate | ~3 hrs |
| **Cyber Threat Intelligence** | IOCs, threat feeds, MITRE ATT&CK framework | ~5 hrs |
| **SIEM Fundamentals** | What SIEMs do, log ingestion, correlation rules | ~4 hrs |
| **Splunk** | Searching, dashboards, SPL query language | ~6 hrs |
| **Elastic Stack (ELK)** | Kibana, Elasticsearch, Logstash, KQL | ~6 hrs |
| **Network Security & Traffic Analysis** | Zeek, Snort, Wireshark deep dive | ~6 hrs |
| **Endpoint Security** | EDR tools, Windows event logs, sysmon | ~5 hrs |
| **Phishing Analysis** (advanced) | Full phishing case workflows | ~4 hrs |
| **DFIR Module** | Artifacts, evidence collection, Windows/Linux forensics | ~7 hrs |
| **Capstone Challenges** | Full attack chain investigations | ~5 hrs |

**Total Estimated: ~51 hours**

## 🔑 Key Skill: MITRE ATT&CK Framework
This is the most important framework in defensive security. Learn to map every attack you see to a tactic + technique. Use [https://attack.mitre.org](https://attack.mitre.org) as a reference constantly.
