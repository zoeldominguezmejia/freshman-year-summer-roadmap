# Phase 2 — Cyber Security 101 Learning Path
**Weeks 3–5 | June 9 – June 29 | ~50 hours | ~17 hrs/week**

---

## 🎯 Goal
Build a broad, practical foundation across both offensive and defensive cybersecurity. This is TryHackMe's flagship beginner path — 54 rooms, 13 modules — designed to replace the old Complete Beginner and Intro to Cyber Security paths.

> 💡 This path integrates red team and blue team content together, giving you context for *why* defenses matter by showing you the attacks first.

---

## 📚 Learning Path: Cyber Security 101
**Link:** https://tryhackme.com/path/outline/cybersecurity101

### Module Breakdown

| Module | Topics | Est. Time |
|--------|--------|-----------|
| **Intro to Cyber Security** | Offense vs defense, careers, lab setup | ~3 hrs |
| **Command Line** | Bash, PowerShell, scripting fundamentals | ~4 hrs |
| **Google Dorking & OSINT** | Passive recon, Google hacks, Shodan | ~3 hrs |
| **Networking** | Protocols deep dive, Wireshark intro | ~5 hrs |
| **Exploitation Basics** | Vulnerability scanning, Metasploit basics | ~6 hrs |
| **Web Hacking** | OWASP Top 10, Burp Suite intro, SQL injection | ~7 hrs |
| **Cryptography** | Hashing, encryption, password cracking | ~4 hrs |
| **Linux Privilege Escalation** | SUID, sudo misconfigs, cron jobs | ~5 hrs |
| **Windows Privilege Escalation** | Windows misconfigs, token abuse | ~5 hrs |
| **Blue Team Fundamentals** | Log analysis, SIEM intro, threat hunting | ~4 hrs |
| **Intro to DFIR** | What DFIR is, artifact types, evidence collection | ~4 hrs |
| **Phishing Analysis** | Email headers, phishing detection | ~3 hrs |
| **Capstone Challenges** | Full-chain practice scenarios | ~4 hrs |

**Total Estimated: ~57 hours (pace yourself)**
