## 🔗 Supplemental Resources
 
- **Networking Visualizer:** [Cisco Packet Tracer (Free)](https://www.netacad.com/courses/packet-tracer) — Build mock LANs to visualize how packets route before diving into labs.
- **Linux Practice Platform:** [OverTheWire: Bandit](https://overthewire.org/wargames/bandit/) — A text-based game that forces you to use the command line to find hidden flags. Great companion to the Linux CLI rooms.
- **Cheat Sheet Tool:** [ExplainShell](https://explainshell.com/) — Paste any Linux command and it breaks down exactly what every flag does.
- **OS Internals Video:** [How the Windows Kernel Works Under the Hood](https://youtu.be/804bA_p8_P0) — Deep conceptual dive into processes, threads, and memory rings.
- **SQL Practice:** [SQLZoo](https://sqlzoo.net/) — Free interactive SQL exercises. Relevant to Module 6 and will come back during Phase 2's SQL injection rooms.
